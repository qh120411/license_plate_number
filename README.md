Step 1: Clone the repository
```bash
git clone <linkrepo>
```
Step 2: Open vscode, Open your folder and create a python enviroment by click terminal -> new terminal
```bash
python -m venv venv
```
Step 3: Activate the Python environment
```bash
venv\Scripts\activate.bat
```
Step 4: Install required framework
``` bash
pip install opencv-python easyocr ultralytics
```
Step 5: Open demo.py and run the script
```bash
python demo.py
```
